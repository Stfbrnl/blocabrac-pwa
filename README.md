# BLOCABRAC PWA

Application Progressive Web App (PWA) pour la salle d'escalade BLOCABRAC.

## 📱 À propos

BLOCABRAC PWA est une application web progressive permettant aux grimpeurs de la salle BLOCABRAC de :
- Consulter les horaires d'ouverture
- Gérer leurs réservations
- Suivre leurs performances
- Accéder à l'application hors ligne

## 🚀 Fonctionnalités

- Progressive Web App (PWA)
- Interface responsive
- Fonctionnement hors ligne
- Installation sur écran d'accueil

## 📋 Prérequis

- Node.js (version x.x.x ou supérieure)
- npm ou yarn

## 🔧 Installation

```bash
# Cloner le repository
git clone https://github.com/Stfbrnl/blocabrac-pwa.git

# Accéder au répertoire
cd blocabrac-pwa

# Installer les dépendances
npm install
```

## 💻 Démarrage

```bash
# Mode développement
npm run dev

# Build pour production
npm run build

# Démarrer le serveur de production
npm start
```
